## 3. 실현 가능성 분석 (내부용)

- **프로젝트 유형**: 복잡한 실시간(WebSocket 양방향 대시보드) + 풀스택 웹앱 + 공공 API 연동 + RBAC/인증 → "주의" 등급(+25% 버퍼)
- **기본 공수 (AI 미반영)**: 기획 15 + 디자인 12 + FE 30 + BE 35 + QA/배포 13 = **약 105 M/D**
- **AI 반영 공수**: 실시간 아키텍처 난이도 고려 보수적 ~48% 절감 → 약 55 M/D, +25% 버퍼 = **약 68 M/D**
- **1인 환산 기간**: 68 × (7/5) ≈ **약 95일** → 90일 소폭 초과(약 6%)
- **클라이언트 예상 기간 vs 산정 기간**: 90일 vs 95일(1인 기준)
- **판정**: 1인 기준 90일은 빠듯하나, **다수 인력 투입 시 디자인↔개발 단계 병행으로 90일 유지 충분히 가능**. 1순위가 "일정 준수"이고 평균 입찰가(28.1M)보다 낮은 가격으로 2순위 "금액" 경쟁력 확보.
- **법인/보증보험 자격 요건**: 충족 가능(법인, 업력 1년 이상, 보증보험 발급 가능).
- **사용자 확정**: 금액 **25,500,000원(예상가 85%)** / 기간 **90일 유지** 선택.

- **핵심 제안 포인트**:
  1. 검증된 실시간 양방향 동기화 경험(CRDT/Yjs → Socket.IO WebSocket) + 상시 접속 끊김 방지·자동 재연결

  4. FE/BE 완전 분리 RESTful + WebSocket으로 2단계(모바일·챗봇) 확장 대비
- **기술 스택**: React 18 + Next.js(SPA) / NestJS + Socket.IO / PostgreSQL + Redis(Pub/Sub) / JWT·2FA·CASL / Firebase FCM / AWS / Docker
